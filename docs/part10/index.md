## 设计思想主线

!!! tip "本编建立的认知基础"
    1. 十大设计模式不依赖 AI 技术——它们是**通用的软件架构智慧**
    2. 性能工程涉及打包、分割、Tree Shaking——**每个决策都影响用户体验**
    3. Source Map 泄露给了我们教训——**构建安全不容忽视**
    4. 分辨原作和补全是读逆向代码的**必备能力**
    5. AI 编程助手的架构选择暗示了**未来方向**
