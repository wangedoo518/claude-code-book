## 设计思想主线

!!! tip "本编建立的认知基础"
    1. MCP 协议是 AI 时代的 USB——**统一接口降低接入成本，释放生态潜力**
    2. 桥接系统让 CLI 的能力"长"到 IDE 里——**三代迭代解决了稳定性和便携性**
    3. 工具服务于 AI 的自主决策，命令服务于用户的直接意图——**边界清晰**
    4. 插件系统在"开放"和"安全"之间走钢丝——**审核、沙箱、权限三管齐下**
