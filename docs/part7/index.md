## 设计思想主线

!!! tip "本编建立的认知基础"
    1. 四层记忆（会话缓冲→短期→长期→团队共识）各解决不同的**"遗忘问题"**
    2. CLAUDE.md 是"教科书"，MEMORY.md 是"笔记本"——**每种"笔记"有不同用途**
    3. 压缩系统有三种策略：手动/定时/紧急——**在保留核心信息的前提下缩减上下文**
    4. 区分实验性代码和正式功能是**高效读源码的必备技能**
